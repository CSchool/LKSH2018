Диме на новый год родители подарили мешок с бесконечным количеством доминошек размером 2 × 1 сантиметров. Также с дня рождения у него остался никуда не приспособленная дубовая доска размером 3 × N сантиметров. Ему вдруг стало интересно, сколькими способами можно замостить эту доску теми доминошками. Мы не спрашиваем почему. Просто ответьте на его вопрос, потому что иначе на следующий праздник он попросит книгу по динамическому программированию.

Входные данные

На вход подаётся одно целое число — N (1 ≤ N ≤ 50)
