В сообщении, состоящем из одних русских букв и пробелов, каждую букву заменили её порядковым номером в русском алфавите (А — 1, Б — 2, ..., Я — 33), а пробел - нулем. Требуется по заданной последовательности цифр найти количество исходных сообщений, из которых она могла получиться.

Входные данные

В первой строке содержится последовательность цифр. Длина последовательности не превышает 100.

Выходные данные

Выведите одно число — количество возможных сообщений. Гарантируется, что это число влезает в беззнаковый 64-битный целый тип данных.

Пример

Входные данные
1025
Выходные данные
4
