Fk — бесконечная последовательность целых чисел, которая удовлетворяет условию Фибоначчи Fk = Fk - 1 + Fk - 2 (для любого целого k). Даны i, Fi, j, Fj, n (i ≠ j). Найти Fn. Пример части последовательности: F - 2 =  - 5, F - 1 = 4, F0 =  - 1, F1 = 3, F2 = 2, F3 = 5, F4 = 7, F5 = 12, F6 = 19

Ограничения:  - 1000 < i, j, n < 1000,  - 2000000000 < Fk < 2000000000 (k = min(i, j, n)... max(i, j, n))

Входные данные

В первой строке находятся числа i, Fi, j, Fj, n.

Выходные данные

Вывести одно число Fn.

Пример

Входные данные
3 5 -1 4 5
Выходные данные
12
