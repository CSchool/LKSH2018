Лесенкой называется набор кубиков в один или несколько слоёв, в котором каждый более верхний слой содержит кубиков меньше, чем нижний.

Подсчитать высоту лесенки из N ступенек.

Входные данные

В первой строке записано число N (1 ≤ N ≤ 100).

Во второй строке через пробел записаны N положительных целых чисел, не превосходящих 109 — высоты ступенек в порядке невозрастания.

Выходные данные

Выведите одно целое число — высоту лесенки.

Пример

Входные данные
3
4 2 0
Выходные данные
4
