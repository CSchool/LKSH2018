Дано N целых чисел A1, A2, ..., AN. Требуется найти количество различных значений сумм вида k1A1 + k2A2 + ... + kNAN, где ki равняется 1 или 0.

Входные данные

В первой строке находится число N, во второй — A1, A2, ..., AN через пробел.

Ограничения: 1 < N < 500, 0 < Ai < 100, все числа целые.

Выходные данные

Вывести одно число — количество различных значений сумм.

Примеры

Входные данные
3
1 1 2
Выходные данные
5
Входные данные
3
1 3 2
Выходные данные
7
Входные данные
5
49 100 98 49 0
Выходные данные
10
